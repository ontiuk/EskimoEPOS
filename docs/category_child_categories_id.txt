Eskimo EPOS Category Import
==============================

2-Way ImpEx of EPOS Caegory data into WooCommerce Product Category

Woocommerce category name -> ShortDescription
Woocommerce category desc -> LongDesription

Process
- create wc category name from short description
- test if already exists
- check if parent is set - should be empty/0
    - get wc category using Eskimo_Category_ID and custom term_meta field 'eskimo_category_id'
    - if set get_term data for parent inc term_id
- create term data: name, description, parent_id, slug autogenerated
- insert term if not already exists
- return list of inserted categories
    - generate a web id for each from option category prefix + term_id
    - update remote category cart IDs with new webID

log progess and return data / errors
